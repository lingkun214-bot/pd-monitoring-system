# PD 局放在线监测系统 HTML 原型

## 打开方式

直接双击本目录下的：

```text
index.html
```

无需安装依赖或启动服务。

## 登录

- 账号：`admin`
- 密码：`12345`
- 登录状态会保存在浏览器本地，刷新页面后仍保持登录。
- 点击“退出登录”可清除登录状态并返回登录页。

## 页面

系统包含登录页，以及以下 8 个主页面：

- 实时监测驾驶舱
- 历史数据分析
- 长周期趋势分析
- 智能诊断中心
- 数据处理
- 报警管理
- 设备管理
- 系统设置

## 关键功能

- AI 水轮发电机背景使用项目内的本地图片资产，不依赖外部网络。
- 历史数据支持组合查询、动态回放及 CSV、JSON、波形 JSON 导出；趋势目标与摘要、图表和导出数据保持联动。
- 智能诊断在执行前保持中性状态；完成后使用同一可追溯诊断上下文生成预览、可打印 HTML 报告，并保留审核人、签名和日期。
- 数据处理支持滤波参数校验、保存，以及相位屏蔽窗的新增、编辑、启停、删除和重叠校验。
- 报警管理支持机组、等级、状态组合筛选，并按“未确认—已派发—已确认—已关闭”流程处置。
- 设备配置、阈值、老化系数、IRIS 基准、演示用户和系统日志均可操作并保存到浏览器 `localStorage`。
- 系统设置支持可取消、失败模拟和重试的硬件自检；系统日志支持筛选、详情和 CSV 导出。
- 顶部报警铃铛显示未确认报警数量；底部状态栏明确显示冻结样例截止时间和“非实时采集”状态。

## 设备导航范围

- 保留设备树：历史数据分析、长周期趋势分析、数据处理。
- 不显示设备树：实时监测、智能诊断、报警管理、设备管理、系统设置。
- 智能诊断使用模块内的机组和通道条件作为诊断目标。

## 原型边界

本项目是纯前端静态原型，无后端服务。当前版本不模拟生产系统已经成功接入身份、数据库、采集或算法服务，也不把页面操作包装成生产成功响应。以下能力明确留待生产环境接入：

NEW-007 说明：当前版本的登录校验、用户、设备配置、阈值、报警处置和系统日志均为前端演示逻辑，部分状态写入当前浏览器 `localStorage`。停用演示用户不会改变固定 `admin` 登录，本地日志可被修改，不能作为生产审计证据。生产环境必须接入后端身份认证、RBAC、会话控制、不可篡改审计、可靠服务端持久化、真实采集接口及统一时间同步；不得直接将本原型作为生产系统部署。

- OIDC 或统一身份认证、密码安全策略、服务端会话和 RBAC 权限控制；用户管理目前只演示资料与启停状态，登录仍固定为 `admin / 12345`。
- 真实传感器、采集卡、采集心跳、硬件自检接口及实时数据流；当前页面只展示截止到 `2025-05-20 14:32:18 UTC+8` 的冻结样例，不生成虚假“刚刚采集”时间。
- 数据库持久化、多用户并发、不可篡改审计日志和服务端报警工作流；`localStorage` 仅用于原型交互连续性。
- 版本化算法服务、模型发布审批、输入数据质量门禁及结果复核；当前确定性结果只用于演示前后端契约，校准缺失或过期时返回“无法判定，需人工复核”。
- 正式 PDF 报告生成、电子签名、文件归档和下载权限；当前下载为可打印 HTML 报告。
- 生产环境 API 的超时、断线重连、权限异常和跨设备数据一致性处理。

## 生产接口保留字段

页面与导出数据使用以下字段作为后续接口契约边界：

- `assetId`：稳定测量对象标识。
- `datasetId`：诊断及导出所绑定的数据集。
- `sampleAsOf`：数据样例截止时间。
- `qualityCode`：数据质量码。
- `calibration`：校准状态、证书、有效期、工程单位、原始单位与不确定度。
- `algorithmVersion`：诊断算法版本。
- `ruleVersion`：阈值判级规则版本。
