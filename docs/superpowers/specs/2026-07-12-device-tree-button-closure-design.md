# 设备树与按钮闭环设计规格

## 目标

为除“实时监测驾驶舱”外的所有业务页面增加统一、可收起的设备树，并逐页补齐按钮操作反馈，使查询、筛选、回放、保存、诊断、导出、派发与关闭等操作形成可验证的完整链路。

## 范围

涉及页面：历史数据分析、长周期趋势分析、智能诊断中心、数据处理、报警管理、设备管理、系统设置。

实时监测驾驶舱沿用现有机组卡片选择方式，不新增设备树。历史数据分析中的“原始脉冲波形查询”面板保留，但删除无实际用途的“关闭 ×”按钮。

## 设备树结构

设备树采用用户选择的 C 方案：可收起侧栏。

- 展开宽度约 220px，展示“厂站 → 机组 → 设备 → 通道”四级结构。
- 收起后保留窄栏，显示当前机组、设备类型和通道摘要，避免设备上下文消失。
- 侧栏提供展开/收起按钮和设备搜索输入框。
- 页面标题下方显示当前选择路径，例如“灵昆水电站 / 3#机组 / 发电机 / A相 UHF”。
- 设备选择保存在统一页面状态中，切换功能页面后继续沿用。
- 点击机组、设备或通道后，右侧数据、图表、条件摘要和操作对象同步刷新，并显示切换反馈。

### 选择模式

- 默认采用单选，点击节点后立即切换当前设备上下文。
- 历史数据分析和报警管理额外提供多选模式，用于跨通道汇总查询。
- 多选模式退出后恢复到最后一次单选的节点。
- 没有可用数据时显示空状态，不保留上一设备的结果造成误解。

### 页面适配

- 历史数据分析：树选择与机组/通道筛选同步，回放和原始波形绑定当前记录与通道。
- 长周期趋势分析：趋势、阈值、老化系数和 IRIS 基准绑定当前机组或设备类型。
- 智能诊断中心：诊断输入和报告对象绑定当前设备、通道。
- 数据处理：滤波参数和处理结果绑定当前通道。
- 报警管理：单选查看设备报警，多选汇总多个通道报警。
- 设备管理：树作为设备拓扑导航，右侧编辑当前设备或通道配置。
- 系统设置：自检、硬件配置和系统日志受设备上下文约束；用户管理保持全局范围，并明确显示“全局设置”。

## 按钮闭环规则

所有可交互按钮必须满足至少一种结果：改变页面状态、更新数据、打开详情、生成输出或给出明确反馈。没有用途的装饰按钮删除。

### 查询与筛选

- 查询后更新表格、图表、统计数量和条件摘要。
- 重置恢复默认条件并重新加载默认结果。
- 条件不完整时不执行查询，提示缺失项。
- 无结果时显示空状态和当前查询条件。

### 回放与详情

- PRPD 播放、暂停和倍速控件反映当前播放状态、进度和速度。
- 回放按钮切换当前记录并更新 PRPD 与原始脉冲数据。
- 详情、预览类按钮打开可关闭的详情面板或弹窗，并保留返回原列表的路径。

### 保存、应用与诊断

- 操作开始后进入短暂处理态并防止重复点击。
- 成功后显示提示并更新受影响的数据或状态。
- 校验失败时指出具体字段或条件，不清空用户已输入内容。

### 导出与打印

- 导出操作显示所选格式、数据范围和生成结果。
- 打印操作打开打印预览反馈，不静默结束。
- 无可导出数据时说明原因并保持当前页面状态。

### 危险操作

- 删除、关闭告警等不可逆操作必须二次确认。
- 确认后更新列表、计数和详情状态；取消则不修改数据。
- 操作完成后提示对象名称和结果。

### 统一反馈

使用轻量提示条展示成功、失败、取消和设备切换消息。提示内容必须包含动作和对象，避免仅显示“操作成功”。

## 状态模型

统一状态至少包含：

- 当前页面；
- 设备树展开状态；
- 当前厂站、机组、设备和通道；
- 历史分析与报警管理的多选通道；
- 每页筛选条件；
- PRPD 回放状态与倍速；
- 当前操作处理态；
- 最近一次反馈消息。

状态只在原型前端内维护，不引入后端或账户级持久化。刷新页面后可恢复默认设备上下文。

## 异常与边界

- 当前节点无数据：显示空状态，禁用依赖数据的按钮。
- 多选通道不兼容：提示冲突并保留已选节点，等待用户调整。
- 重复提交：处理态期间禁用同一操作按钮。
- 危险操作取消：关闭确认框并保持原数据。
- 页面切换：保留设备上下文，清理仅属于上一页面的临时弹窗和处理态。

## 验收与测试

自动化结构与行为测试必须覆盖：

1. 7 个非实时页面均接入统一设备树容器；
2. 设备树可展开、收起、搜索和单选；
3. 历史分析与报警管理支持多选；
4. 设备选择会更新路径摘要和页面数据状态；
5. 查询、重置、回放、详情、保存、诊断、导出、打印和报警操作均产生可见结果；
6. 删除和关闭操作存在确认与取消路径；
7. 原始脉冲波形面板不再包含“关闭 ×”按钮；
8. 处理态可防止重复操作；
9. 现有登录、阈值、报警状态同步等测试继续通过。

浏览器验收逐页检查：无意外横向滚动、侧栏收起后图表宽度恢复、设备路径清晰、按钮不出现无响应、操作后能够返回稳定且可继续使用的页面状态。

## 非目标

- 不接入真实设备数据或后端接口；
- 不改变实时监测驾驶舱的现有机组卡片导航；
- 不重做已确认的整体视觉风格；
- 不在本次范围内实现账户级设备权限。
